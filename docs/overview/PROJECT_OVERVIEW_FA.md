\# QuantLab AI — معرفی پروژه



\## 1. هدف پروژه



QuantLab AI یک پلتفرم Fullstack برای تحلیل داده‌های مالی بازار ASX است.



هدف پروژه این است که یک اپلیکیشن حرفه‌ای تحلیل داده مالی بسازیم که توانایی‌های زیر را نشان دهد:



\- توسعه Backend با Python

\- طراحی API با FastAPI

\- توسعه Frontend با React

\- ساخت ETL Pipeline

\- تحلیل داده‌های سری زمانی مالی

\- محاسبه معیارهای ریسک

\- Backtesting

\- تحلیل پرتفوی

\- معماری نرم‌افزار تمیز و قابل توسعه



این پروژه هم برای رزومه و نمونه‌کار ارزشمند است و هم می‌تواند به‌عنوان یک نمونه اولیه برای تحلیل بازار استفاده شود.



\---



\## 2. اهداف اصلی



اهداف اصلی پروژه عبارت‌اند از:



1\. ساخت یک معماری Fullstack تمیز.

2\. دریافت داده‌های بازار ASX.

3\. ذخیره داده‌های مالی در دیتابیس محلی.

4\. تحلیل تغییرات قیمت و ریسک.

5\. نمایش روندهای بازار با نمودار.

6\. ساخت سرویس‌های Backend قابل استفاده مجدد.

7\. ساخت داشبورد تحلیلی حرفه‌ای.

8\. مستندسازی کامل تمام مراحل.

\## 5. ویژگی‌های نسخه MVP



\### جست‌وجو و بررسی بازار ASX



اپلیکیشن باید بتواند شرکت‌های ASX را نمایش دهد و امکان فیلتر کردن بر اساس موارد زیر را بدهد:



\- نماد سهم

\- صنعت

\- ارزش بازار

\- محدوده قیمت

\- نوسان



\### نمودارهای مالی



اپلیکیشن باید نمودارهای زیر را نمایش دهد:



\- نمودار قیمت

\- نمودار حجم معاملات

\- میانگین متحرک

\- RSI

\- MACD



\### ETL Pipeline



پایپ‌لاین ETL باید این کارها را انجام دهد:



1\. دریافت داده از Yahoo Finance.

2\. پاک‌سازی و آماده‌سازی داده خام.

3\. ذخیره داده در SQLite.



\### تحلیل ریسک



سیستم باید معیارهای زیر را محاسبه کند:



\- Volatility

\- Sharpe Ratio

\- Maximum Drawdown

\- Rolling Volatility

\- Correlation Metrics



\### Backtesting



سیستم باید استراتژی‌های ساده زیر را پشتیبانی کند:



\- Buy and Hold

\- SMA Crossover

\- RSI Strategy



\### تحلیل پرتفوی



کاربر باید بتواند:



\- چند سهم ASX را انتخاب کند

\- وزن هر سهم را مشخص کند

\- عملکرد پرتفوی را بررسی کند

\- تمرکز صنعتی پرتفوی را ببیند
